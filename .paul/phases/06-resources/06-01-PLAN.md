# Phase 06 — Resources Hub

## Goal
Give students a curated library of resources organized by category (Faith, Fitness, Academics, Life Skills). Each resource card links out to an external URL. Resources are seeded via migration and managed in Supabase.

## Deliverables

| File | Status |
|------|--------|
| `supabase/migrations/004_resources.sql` | ✓ Done |
| `lib/resources.ts` | ✓ Done |
| `app/(tabs)/resources.tsx` | ✓ Done |

## DB Schema

```sql
resources (
  id            uuid PK default gen_random_uuid()
  title         text NOT NULL
  description   text NOT NULL
  url           text NOT NULL
  category      text ('faith','fitness','academics','life')
  resource_type text ('article','video','podcast','tool')
  active        boolean default true
  created_at    timestamptz default now()
)
```

RLS: authenticated and anonymous users can SELECT active resources; no INSERT/UPDATE/DELETE from client.

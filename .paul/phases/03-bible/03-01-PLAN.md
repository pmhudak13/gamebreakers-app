# Phase 03 — Bible Reader

## Status: COMPLETE

## Goal
Integrate API.Bible (KJV, free tier) with a full book/chapter/verse reading experience styled in GBA navy.

## Deliverables

| File | Purpose |
|------|---------|
| `lib/bible.ts` | API.Bible client — getBooks, getChapters, getChapterVerses + HTML verse parser |

### Modified Files
| File | Change |
|------|--------|
| `app/(tabs)/bible.tsx` | Full Bible reader replacing the Coming Soon placeholder |

## Feature Details
- **Bible**: KJV (`de4e12af7f28f599-01`) via API.Bible free tier
- **Book picker**: Full-screen modal with all 66 books, defaults to John
- **Chapter selector**: Horizontal scroll of chapter numbers, active chapter highlighted white
- **Verse display**: Verse number (gray) + text (white) on dark navy background
- **Error states**: Helpful message if API key missing or network fails
- **Loading state**: Spinner while fetching

## Env Var Required
```
EXPO_PUBLIC_BIBLE_API_KEY=your-key-from-scripture.api.bible
```
Get a free key at https://scripture.api.bible (sign up → create app → copy key)

# Phase 04 — Devotionals

## Goal
Display daily and weekly devotional content stored in Supabase. Users browse a filterable list and tap any card to read the full devotional with scripture context.

## Deliverables

| File | Status |
|------|--------|
| `supabase/migrations/002_devotionals.sql` | ✓ Done |
| `lib/devotionals.ts` | ✓ Done |
| `app/(tabs)/devotionals.tsx` | ✓ Done |

## DB Schema

```sql
devotionals (
  id             uuid PK default gen_random_uuid()
  title          text NOT NULL
  body           text NOT NULL
  scripture_ref  text
  scripture_text text
  type           text CHECK (type IN ('daily','weekly')) default 'daily'
  published_at   timestamptz default now()
  created_at     timestamptz default now()
)
```

RLS: authenticated users can SELECT. INSERT/UPDATE via SQL editor only (admin panel deferred to Phase 09).

## UI Features
- Navy header with subtitle "Faith fuel for the grind"

- FlatList of cards: type badge, date, title, scripture ref, excerpt
- Pull-to-refresh
- Tap card → pageSheet modal with full content

  - Body split into paragraphs
- Empty state, error state with retry button, loading spinner

## Pre-Launch Checklist (Manual Steps)

### EAS Setup (one-time)
```bash
npm install -g eas-cli
eas login            # log in with Kevin's Expo account
eas init             # links the project, populates extra.eas.projectId in app.json
```

### Environment Variables (EAS secrets)
Set these in the EAS dashboard (expo.dev) under project → Secrets:
- `EXPO_PUBLIC_SUPABASE_URL`
- `EXPO_PUBLIC_SUPABASE_ANON_KEY`
- `EXPO_PUBLIC_BIBLE_API_KEY`

### Build & Submit
```bash
# Dev build (installs on device via Expo Go alternative)
npm run build:dev

# Preview build (APK for internal testers)
npm run build:preview

# Production build (App Store + Play Store)
npm run build:prod

# Submit to stores (after production build)
npm run submit:ios
npm run submit:android
```

### App Store Assets Needed
- [ ] App icon 1024×1024 (iOS App Store)
- [ ] Screenshots: iPhone 6.5" (1242×2688), iPhone 5.5" (1242×2208)
- [ ] Screenshots: iPad 12.9" (2048×2732)
- [ ] Android screenshots: phone (1080×1920+), tablet optional
- [ ] Short description (80 chars): "Faith. Fitness. Future. The student-athlete app."
- [ ] Full description (4000 chars max)
- [ ] Privacy policy URL (required by both stores)
- [ ] Support URL

## DB Schema

```sql
tutors (
  id            uuid PK default gen_random_uuid()
  name          text NOT NULL
  bio           text NOT NULL
  avatar_url    text
  subjects      text[] NOT NULL
  grade_levels  text[] NOT NULL
  availability  text[] NOT NULL   -- e.g. ['Mon', 'Wed', 'Fri']
  rate_type     text CHECK IN ('free', 'paid')
  rate_note     text              -- e.g. '$20/hr', 'Free for teammates'
  contact_email text
  active        boolean default true
  created_at    timestamptz default now()
)
```

RLS: authenticated users can SELECT active tutors; no client INSERT/UPDATE/DELETE.

## Route Logic
```
no session          ->  /(auth)/login
session, no profile ->  /profile-setup
session + profile   ->  /(tabs)
```

## Database Setup (manual step)
Run `supabase/migrations/001_profiles.sql` in the Supabase SQL editor.
Make sure your Supabase project has email auth enabled (Auth -> Providers -> Email).

## Profile Schema
```sql
profiles(
  id uuid PK -> auth.users
  name text NOT NULL
  school text
  sport text
  role text CHECK IN ('student','tutor','coach') DEFAULT 'student'
  avatar_url text
  created_at timestamptz
)
```
